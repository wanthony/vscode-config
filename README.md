# Usage

```
cd ~/Library/Application\ Support/Code/User

# Settings
mv settings.json _settings.json
ln -s path/to/vscode-config/settings.json ./settings.json

# Keybindings
mv keybindings.json _keybindings.json
ln -s path/to/vscode-config/keybindings.json ./keybindings.json

# Snippets
mv snippets _snippets
ln -s path/to/vscode-config/snippets ./snippets
```
